Welcome to the fan-made Final Fantasy XIV character search! Begin by searching your character and selecting your server to pull your character's information.

Purpose



The purpose of this website is to pull information from the Final Fantasy XIV fan-made API including job/class information, levels, and an avatar and using a secondary API to call one of the character's favorite jokes. They love to bring these up in taverns for some reason.
